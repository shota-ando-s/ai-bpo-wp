## はじめに

AIを使った業務改善は、もはや大企業だけの話ではありません。

## ステップ1: 繰り返し作業を洗い出す

まず、毎日・毎週繰り返している作業をリストアップします。

- メールの定型返信
- データの集計・転記
- 議事録の作成

## ステップ2: AIツールを試す

洗い出した作業に対して、適したAIツールを選びます。

| 作業 | ツール |
|------|--------|
| 文章作成 | ChatGPT / Claude |
| データ分析 | ChatGPT Code Interpreter |
| 画像生成 | Midjourney |

## まとめ

小さな作業から始めて、少しずつ自動化の範囲を広げていきましょう。
