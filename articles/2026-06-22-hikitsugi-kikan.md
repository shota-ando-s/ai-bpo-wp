退職日が近づいているのに、後任への引き継ぎがまったく進んでいない。日々の業務に追われ、気づけば残り数週間しかない——そんな状況に直面していませんか。退職や異動のたびに「引き継ぎが間に合わない」と慌てるのは、適切な**引き継ぎ期間**を見積もり、逆算してスケジュールを組めていないことが大きな原因です。本記事では、退職時の引き継ぎ期間の目安と、現実的なスケジュールの立て方を解説します。

> 本記事は [業務引き継ぎ完全ガイド](/gyomu-hikitsugi-guide) のクラスター記事です。

## 引き継ぎ期間の目安はどれくらい？

引き継ぎに必要な期間は、担当業務の複雑さや責任範囲によって大きく変わります。単純な定型業務であれば数週間で足りる一方、属人化が進んだ専門業務や、決裁権を持つ役職者の引き継ぎには2ヶ月程度を見込んでおきたいところです。

以下はあくまで一般的な目安です。実際の必要期間は、業務量・後任のスキル・繁忙期かどうかなどで前後します。

| 業務の複雑さ・役職 | 引き継ぎ期間の目安 |
| --- | --- |
| 定型的な一般事務・ルーティン業務 | 2週間〜3週間 |
| 複数の取引先や案件を抱える担当者 | 1ヶ月程度 |
| 専門知識・属人化した業務の担当者 | 1ヶ月〜2ヶ月 |
| 管理職・決裁権を持つ役職者 | 1ヶ月半〜2ヶ月以上 |

ここで重要なのは、**「目安」はあくまで目安だ**ということです。引き継ぎは「教えて終わり」ではなく、後任が一人で回せるようになって初めて完了します。後任が実務を試し、つまずいた点を質問できる「並走期間」を含めて見積もり、上記より長めに確保しておくと安全です。何から始めればよいか迷う場合は、[引き継ぎの進め方](/hikitsugi-susumekata)も参考になります。

## 退職予告と引き継ぎ期間の関係

「いつまでに退職を申し出れば、引き継ぎ期間を確保できるのか」は多くの人が気にする点です。

一般に、退職の意思表示の時期については民法上のルールや、各社の就業規則による定めがあるとされています。就業規則で「退職の○ヶ月前までに申し出る」といった規定を設けている企業も多く、まずは自社の就業規則を確認するのが確実です。いつまでに申し出るべきかは契約形態や個別事情によって異なるため、判断に迷う場合は人事担当者や専門家に相談しましょう。

実務上のポイントは、**退職予告のタイミングと、引き継ぎに使える実働日数は別物**だということです。たとえば退職の1ヶ月前に申し出たとしても、有給休暇の消化や繁忙期の通常業務が重なれば、実際に引き継ぎに充てられる時間はその半分以下になることも珍しくありません。予告期間ではなく「引き継ぎに割ける実働時間」を基準に逆算することが、間に合わせるコツです。

退職にともなう引き継ぎを意図的に行わないケースのリスクについては、[引き継ぎをしない退職のリスク](/hikitsugi-shinai-taishoku)も参考になります。

## 退職日から逆算するスケジュールの立て方

引き継ぎは「できることからやる」のではなく、退職日（または引き継ぎ完了日）から逆算して組み立てると、抜け漏れと終盤の混乱を防げます。退職日を起点に、後ろから時期ごとのタスクを割り当てましょう。

| 時期（退職日からの逆算） | 主にやること |
| --- | --- |
| 約1〜2ヶ月前 | 担当業務の棚卸し・リスト化、後任の決定、引き継ぎ範囲の確定 |
| 約3〜4週間前 | 引き継ぎ資料・マニュアルの作成、関係者・取引先への連絡準備 |
| 約2週間前 | 後任への説明・OJT、実務に同席してもらう並走の開始 |
| 約1週間前 | 後任が主担当として実務を行い、退職者がフォローに回る |
| 最終日まで | 残課題の文書化、緊急連絡先・パスワード等の最終確認、引き継ぎ完了の確認 |

最初の「棚卸し」がスケジュール全体の土台になります。自分が担当している業務を漏れなく洗い出し、それぞれの頻度・難易度・関係者を一覧化しておくと、どこに時間をかけるべきかが見えてきます。棚卸しを後回しにすると、終盤になって「実はこの業務も引き継ぐ必要があった」と発覚し、スケジュールが破綻しがちです。

## 引き継ぎ期間が足りないときの対処法

退職の申し出が直前になった、繁忙期と重なった、後任の着任が遅れた——理由はさまざまですが、十分な期間を確保できないケースは実際に多くあります。期間が足りないときは、次の3つを意識すると被害を最小限にできます。

**1. 優先順位をつける**
すべてを完璧に引き継ごうとすると、かえって何も終わりません。「止まると会社が困る業務」「期日が決まっている業務」を最優先にし、頻度の低い業務やイレギュラー対応は後回し、あるいは文書化のみにとどめます。

**2. 口頭ではなく文書で残す**
時間がないときほど、対面のレクチャーは最小限にして文書化を優先しましょう。手順書・チェックリスト・連絡先一覧が残っていれば、後任は退職者がいなくなった後でも自力で対応できます。口頭だけの引き継ぎは、退職と同時に情報が消えてしまうのが最大のリスクです。

**3. 外部委託を検討する**
社内に引き継ぎ先がいない、または期間内に育成が間に合わない場合は、業務の一部を外部に委託する選択肢もあります。属人化していた業務を外部の専門サービスに切り出すことで、退職による業務停止を防ぎ、社内の負担も平準化できます。

実際、人手不足や後継者不在を背景にした「退職型」の倒産は増加傾向にあります。帝国データバンクの調査によれば、従業員の退職を要因とする倒産は2024年に87件発生し、過去最多となりました（[出典：帝国データバンク](https://www.tdb.co.jp/report/economic/20250309_taisyoku/)）。引き継ぎが回らないことは、個人の問題ではなく企業の存続にも関わる経営課題になりつつあります。

## 異動・産休育休では期間の考え方が変わる

引き継ぎ期間の組み立て方は、退職か、異動か、産休育休かによって変わります。

人事異動の場合は、退職と違って引き継ぎ後も社内で連絡が取れることが多く、緊急時のフォローが効きやすい一方、辞令から異動日までの期間が短く設定されがちです。短期間で確実に引き継ぐコツは[人事異動の引き継ぎ](/idou-hikitsugi)で解説しています。

産休・育休の場合は、復職を前提とするため「一時的に業務を預ける」性質が強く、復帰後に元の担当へ戻せるよう記録を整えておくことが重要です。体調や出産時期の都合で予定より早く引き継ぎが必要になる可能性も見込み、早めに着手すると安心です。詳しくは[産休・育休の引き継ぎ](/sankyu-hikitsugi)をご覧ください。

なお参考として、厚生労働省の令和6年雇用動向調査では離職率は14.2%とされています。毎年一定数の人材が入れ替わることを踏まえれば、引き継ぎは特別な出来事ではなく、年間スケジュールに織り込んでおくべき定常業務だといえます。

## よくある質問（FAQ）

### 退職時の引き継ぎ期間は最低どれくらい必要ですか？
定型業務なら2〜3週間、複数の取引先や専門業務を抱える場合は1ヶ月以上を目安にするのが一般的です。ただし後任のスキルや繁忙期によって変わるため、並走期間を含めて余裕を持って見積もることをおすすめします。

### 引き継ぎ期間に有給消化は含めて考えるべきですか？
有給休暇を消化すると、その分だけ実際に引き継ぎへ充てられる日数は減ります。退職予告から退職日までの暦日ではなく、有給消化や通常業務を差し引いた「実働時間」を基準に逆算するのが現実的です。

### 引き継ぎが間に合わないときは何を優先すべきですか？
「止まると会社が困る業務」「期日が決まっている業務」を最優先にしてください。頻度の低い業務は文書化にとどめ、口頭での詳細説明よりも手順書・チェックリストを残すことを優先すると、後任が後から自力で対応できます。

### 退職は何ヶ月前に伝えれば引き継ぎ期間を確保できますか？
退職の申し出時期は就業規則や契約形態によって異なるため、まずは自社の就業規則を確認してください。引き継ぎ期間を十分に取るには、就業規則の規定よりも早めに、業務量に見合った余裕をもって申し出るのが安全です。

### 後任が決まっていない場合はどう引き継げばよいですか？
後任が未定でも、業務の棚卸しと文書化は先行して進められます。手順書を整備しておけば、後任が決まった時点で短期間でキャッチアップできます。社内に引き継ぎ先がいない場合は、外部委託も選択肢になります。

### 異動と退職で引き継ぎ期間の考え方は違いますか？
異動は引き継ぎ後も社内でフォローしやすい反面、準備期間が短い傾向があります。退職は退職後の連絡が取りづらいため、文書化の重要度がより高くなります。状況に応じて優先順位を変えることが大切です。

## まとめ

- 引き継ぎ期間の目安は、定型業務で2〜3週間、専門業務や役職者で1〜2ヶ月。あくまで目安として、並走期間を含めて余裕を持って見積もる。
- 退職予告期間ではなく、有給消化や通常業務を差し引いた「引き継ぎに割ける実働時間」を基準に逆算する。
- スケジュールは退職日から逆算し、まず業務の棚卸しから着手する。
- 期間が足りないときは、優先順位づけ・文書化・外部委託の3つで被害を最小化する。
