## 執筆ルール（差別化ポイント）
- 中小企業視点で書く（競合は大企業・ベンダー目線が多い）
- ROI試算・費用の具体的な数値を必ず入れる（競合にない）
- 公的データ（矢野経済研究所・IDC・総務省白書）を引用する（競合ゼロ本が引用していない）
- 「向いていない企業・失敗事例」を正直に書く（信頼性差別化）
- FAQは7問以上（PAAとAIO対策）
